# 酒店管理系统简介 java861

## 项目概述

酒店管理系统是一个功能完备的系统，采用Spring Boot和Vue前后端分离的架构设计，搭配MySQL数据库。系统有效支撑管理员与用户两大角色的功能需求。

## 技术栈

- 前端：Vue
- 后端：Spring Boot
- 数据库：MySQL

## 功能模块

### 管理员功能

- **首页概览**：全局数据监控
- **用户管理**：编辑、删除用户信息
- **房间管理**：增删改查房源信息
- **评论监控**：查看用户反馈
- **订单处理**：安排入住、作废订单，查看订单列表

### 用户功能

- **登录与注册**：便捷的用户认证流程
- **房间预订**：实时查看并预订房间
- **订单列表**：随时查看预订状态

## 系统角色

- 管理员
- 普通用户

## 运行环境

- 前端：现代浏览器
- 后端：Java 1.8+
- 数据库：MySQL 5.7+
